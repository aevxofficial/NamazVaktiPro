# NamazVaktiPro
Namaz Vakti Pro App

## Özellikler

- Src layout, Oop Yapısı
- Basit Tasarım
- Mevcut Vakit & Kalan Vakit & Tarih & Saat Göstergesi
- Diyanet Api
- 81 il-ilçe ezan vakti
- Hatırlatıcı bildirim
- 2 Saat, 1 Saat, 30 Dakika, 15 Dakika ve vakit giriş bildirimi
- Namaz kıl butonuna basıldığında bildirimin sonraki vakte kadar gelmemesi
- Sabah vaktinden sonra bildirim gelmez (Öğle vaktinin girişi hariç)
